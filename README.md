# nodejs-boilerplate

This project objective is being a boilerplate to create as many Nodejs projects as I want.

# Step-by-step

-   Inside the project, like: cd nodjs-boilerplate
-   npm i or yarn
-   npm start or yarn start

## Features

-   [x] Controllers, Middlewares and Models
-   [x] typescript
-   [x] MongoDB
-   [x] bcrypt
-   [x] jwt
-   [x] prettier
